Please follow the following CSS format:
[[style name]] {
	float, display, position, top, right, left, bottom, z-index
	margin, padding, border, width, min-width, height, min-height
	overflow
	background
	box-shadow
	font, color, text-align
	}
1. Indent children elements
2. Use shorthand styles except when overriding another style.
	Shorthand -> margin: 0 0 10px; Override -> margin-bottom: 5px;
3. Add comments where appropriate
4. Use all caps for hex colors, for example #CCDDEE
5. Group "OVERRIDES", "SHIMS", and "NO-JS"
