MeshDataImport is an SSIS project for importing Mesh data into a database from an XML and XSD source.
Before using, ensure that the paths to the XML and XSD files are pointing to the correct versions of the files you wish to use.
The tables will be created automatically if they do not exist.

Trang is an open source program (written in Java) used to convert DTD to XSD (see the example bat file for usage).
Mesh does not currently supply XSD files - when importing data from a new XML file, compare the DTD in source control with the up to date Mesh DTD - if there are changes, generate a new XSD file and add to source control.

The C# class file was generated from xsd.exe (provided by Microsoft) using the XSD generated by Trang.
Generating XSD using xsd.exe is possible, however it produces inferior results in this particular case.
